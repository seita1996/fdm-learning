# 関数型ドメインモデリング（FDM）入門（TypeScript + neverthrow）

`slides.md` にSlidevスライドがあります。

## 起動

- `pnpm install`
- `pnpm dev`
- <http://localhost:3030> を開く

## ビルド/エクスポート

- `pnpm build`
- `pnpm export`

## メモ

- Effectライブラリとして `neverthrow`（`Result` / `ResultAsync`）を前提にしています。
- スライド内のコード例は `snippets/fdm/` に置いてあります（Slidevの `<<< @/...` で取り込み）。
